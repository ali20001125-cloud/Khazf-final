# خزف — v5 (تجربة المستخدم)

Next.js 15 · GSAP + ScrollTrigger · Lenis · Tailwind v4 · RTL · ١٩ صفحة

## بنية التنقّل (النهائية المتفق عليها)
- **هيدر**: Top Bar للحملات (يختفي عند التمرير) · ينكمش 80←64px · موبايل: قائمة/لوجو/بحث/حقيبة · ديسكتوب: المتجر، الوصفات، المدونة، من نحن + بحث/مفضلة/حساب/حقيبة
- **القائمة**: مجمّعة بفواصل (متجر ← محتوى ← شركة ← حساب)، بلا أيقونات ولا إيموجي
- **بحث Overlay**: اقتراحات حية أثناء الكتابة + روابط سريعة

## المتجر الموحّد (PLP)
صفحة واحدة، الرابط يقود كل شي: `/products/?cat=espresso&type=موزّع&sort=best`
- تبويبات: الكل / القهوة / الإسبريسو / التقطير (+ بناء البوكس)
- فلاتر فرعية حسب القسم (القهوة بالدولة، الأدوات بالنوع — والمنتج يظهر بأكثر من قسم)
- Breadcrumb + شريط النتائج (العدد + الفرز: الأحدث/الأكثر مبيعاً/السعر)

## جديد v5
- **الوضع الليلي** 🌙: ثلاثي (فاتح/النظام/داكن) — زر بالهيدر يتنقّل + مبدّل بالقائمة. مبني بالكامل على Design Tokens (بلوك `.dark` واحد بـ globals.css يقلب الموقع كله)، لوحة داكنة دافئة مو سوداء، بلا وميض تحميل (سكربت بالـ head)، ويتبع الجهاز تلقائياً بوضع النظام.
- **الوصفات `/recipes/`** (بدل مختبر خزف): حاسبة تحضير — اختر الطريقة (إسبريسو/V60/كيمكس/أيروبرس/فرنش)، حرّك سلايدر كمية البن، وكل الأرقام (ماء/حرارة/Bloom/صبّات/وقت) تتعدّل حياً → **ابدأ التحضير** يفتح مؤقتاً كبيراً بتعليمات كل مرحلة. بلا اختيار قهوة أولاً.
- **المدونة = مركز المحتوى**: دليل الطحن ودليل الأدوات صارا مقالين مرئيين داخلها + مقالان جدد (اختيار المطحنة، V60 vs كاليتا) + فلترة بالوسوم. التقسيم النهائي: المتجر للشراء · الوصفات للتحضير · المدونة للتعلّم.
- **تغليف هدية + رسالة** بصفحة الدفع (مجاناً، تظهر بالملخص).
- **نبّهني عند التوفر**: غلاية رقبة الإوزة نافدة للعرض — جرس بالبطاقة، وإيميل بصفحة المنتج.
